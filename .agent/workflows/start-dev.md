# Start Development Server

## Steps

// turbo
1. Navigate to the project root and start the development server:
```bash
npm run dev
```

2. The app will be available at http://localhost:3003

3. Hot reload is enabled - changes to files will automatically refresh the browser

## Notes
- Uses Turbopack for fast compilation
- Ensure `.env.local` is configured before starting
